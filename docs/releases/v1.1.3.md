# JellyNews v1.1.3 — Notes de release

Date : 2026-07-07

## Résumé

JellyNews v1.1.3 remplace la mascotte d’accueil par la version corrigée fournie, afin de supprimer le défaut de détourage visible sur l’enveloppe volante.

## Nouveautés

- Asset `app/static/brand/jellynews-mascot.png` remplacé par le PNG corrigé.
- Cache-buster applicatif passé en `1.1.3` pour forcer le rechargement de l’image et des assets versionnés.
- Aucun changement fonctionnel côté backend, configuration ou flux d’envoi.

## Validation conseillée

1. Ouvrir l’accueil administrateur.
2. Vérifier que la mascotte apparaît correctement dans le héros.
3. Vérifier que l’enveloppe volante derrière la mascotte ne présente plus de trou transparent ou de mauvais détourage.
4. Purger le cache navigateur/proxy si une ancienne image reste affichée malgré `?v=1.1.3`.

## Rollback

Revenir à l’image ou au commit v1.1.2, puis purger le cache navigateur/proxy si `jellynews-mascot.png?v=1.1.3` ou `style.css?v=1.1.3` ont déjà été servis.
